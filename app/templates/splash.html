Learn about your classmates!

Yalies is a website and mobile app (WIP) that helps you obtain data on the Yale student body. It regularly aggregates and cleans data from multiple sources, including the Yale Face Book and Directory, in order to provide the most useful information possible.

Unlike any existing Yale student information services, Yalies also provides a powerful and flexible API to make fetching student data a breeze. HTML parsing the Yale Face Book can become a thing of the past. Obtain an API token by logging in.

For questions, comments, bug reports, or suggestions, contact erik.boesen@yale.edu.
